# pixiv_bookmark_downloader

Use python to download all your pixiv bookmarks (not include GIF)

使用python下载您的pixiv收藏 (不包括动图)

---
## 环境

Python 3.6 (32bit)

## 使用说明

1. 您需要安装以下库：

- json

- urllib

- time

- os

- re

2. 在 `pixiv.py` 中144-154行添加相应数据，包括

- 下载路径

- 账号Cookie

- 代理（可选）

3. 在当前目录启动cmd，输入 `python pixiv.py` 即可开始下载
